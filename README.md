# Better Than Words

Better Than Words is a simple ESL writing tool that helps students replace overused words with stronger vocabulary.

## How it works
Students type a word they feel they use too often (like "very", "good", or "big").

The app then provides:
- Stronger vocabulary alternatives
- Definitions
- Word forms
- Example sentences showing correct usage

## Purpose
This tool helps ESL learners expand vocabulary, improve writing clarity, and become more confident using varied language.
